# 浏览器长截图小技巧

## 前言

当遇到一篇文章很长，需要长截图的时候，一般人可能会想要找插件，其实浏览器自带的截图功能已经足够强大了。

## 实操

1. F12 打开控制台，找到自己要长截图的元素，这里以 [Vue.js官网](https://cn.vuejs.org/api/application.html#createapp) 为例
2. 右键，选择 【捕获节点屏幕截图】
   
   ![右键选择](https://pic1.imgdb.cn/item/6940d4384a4e4213d00c4875.png)

3. 成功下载
   
   ![成功下载](https://pic1.imgdb.cn/item/6940d48a4a4e4213d00c4e6d.png)

查看它的截图效果，能看到样式、布局排版等还是挺精细的，截图完之后就能用在自己需要的地方了。
