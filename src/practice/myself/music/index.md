# 刀刀音乐

## 项目地址

[刀刀音乐](https://music.duyidao.cn/)

## 项目简介

刀刀音乐是一个基于 Vue3 开发的音乐播放器，它具有以下特点：

- 界面简洁美观，易于使用
- 支持歌词显示，并自动滚动
- 支持多种播放模式，包括单曲循环、随机播放、顺序播放等
