# 刀刀小站

## 网址

[刀刀小站](https://duyidao.github.io/blogweb/#/)

## 项目搭建

1. 创建项目
   
   ```shell
   npm init vite@latest
   ```
    
2. 安装依赖
   
   ::: code-group
   ```shell [npm]
   npm install vue-router less less-loader
   ```
   ```shell [pnpm]
   pnpm install vue-router less less-loader
   ```
   ```shell [yarn]
   yarn add vue-router less less-loader
   ```
   :::
   
3. 启动项目
   ::: code-group
   ```shell [npm]
   npm run dev
   ```
   ```shell [pnpm]
   pnpm dev
   ```
   ```shell [yarn]
   yarn dev
   ```
   :::
   
4. 打包项目
   ::: code-group
   ```shell [npm]
   npm run build
   ```
   ```shell [pnpm]
   pnpm build
   ```
   ```shell [yarn]
   yarn build
   ```
   :::
